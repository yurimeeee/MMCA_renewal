## 👀프로젝트 소개
* React를 활용한 반응형 웹사이트 리뉴얼.
* 구직용 포트폴리오 사이트로 국립현대미술관과는 무관함을 안내 드립니다.
* 사용된 이미지와 내용의 저작권은 국립현대미술관에 있으므로, 무단으로 사용할 수 없습니다.
</br>

## 🗓️개발기간
